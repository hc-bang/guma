# USER_GUIDE (사용자 가이드)

GUMA™를 효율적으로 사용하기 위한 기능 설명 가이드입니다.

---

## 🔍 프로젝트 개요
GUMA™는 간결하고 빠른 나만의 브라우저 시작 페이지입니다. 북마크, 문서 뷰어, 유용한 도구들을 한곳에서 관리할 수 있습니다.

---

## ✨ 주요 기능 사용법

### 1. 검색 엔진 활용
메인 화면의 검색바 좌측 아이콘을 클릭하여 검색 엔진을 전환할 수 있습니다. (네이버, 구글, 유튜브, GitHub 등 지원)

### 2. 바로가기 그리드 (Shortcuts)
- **추가:** 그리드 빈 공간의 `+` 버튼을 눌러 추가합니다.
- **삭제:** 아이콘을 **우클릭**하여 삭제할 수 있습니다.
- **정렬:** 환경설정 페이지에서 직접 순서를 편집하거나 초기화할 수 있습니다.

### 3. 상단 북마크 바
- 상단 바의 링크를 클릭하여 자주 방문하는 사이트로 즉시 이동합니다.
- 드롭다운 폴더 형식의 중첩 메뉴를 지원하여 깔끔한 정리가 가능합니다.

### 4. 다크 및 라이트 모드
- 우측 상단의 테마 보름달/해 아이콘을 클릭하여 테마를 전환합니다.
- 시스템 설정에 따라 자동으로 초기값이 설정되며, 선택한 테마는 브라우저에 저장됩니다.

### 5. 문서 뷰어 및 자료실
- **문서:** 좌측 슬라이드 메뉴의 '문서' 아이콘을 클릭하여 기술 문서 및 가이드를 읽을 수 있습니다.
- **자료실:** 각종 자료를 트리 구조로 탐색하고 다운로드할 수 있습니다.

---

## ⚙️ 사용자 맞춤 설정 (환경설정)
슬라이드 메뉴의 **환경설정** 페이지에서 아래 작업을 수행할 수 있습니다.
- 북마크 및 바로가기 데이터 GUI 기반 편집
- 설정 데이터 내보내기 (JSON 백업) 및 불러오기 (복원)
- 모든 설정 데이터를 초기 상태로 리셋

---

## 💡 팁 및 주의사항
- 검색 시 `Enter` 키를 눌러 기본 검색 엔진으로 빠르게 검색하세요.
- 환경설정에서 저장한 내용은 브라우저의 `localStorage`에 저장되므로, 브라우저를 초기화하면 설정이 사라질 수 있습니다. 중요한 설정은 백업 기능을 이용해 수동으로 저장해 두세요.
